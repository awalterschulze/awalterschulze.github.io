Open Source

GoGoProtobuf: Protocol Buffers for Go with Gadgets.

A fork of Google's protocol buffer generator and library for Go. GoGoProtobuf is faster and more user friendly. It has received over 1000 stars and is used by Docker, Dropbox, Kubernetes, etcd, to name but a few.

GoDerive: Derives and generates golang functions from input types. These include monadic and concurrent functions. It has received over 400 stars, because it trended on github, was published in the golangweekly newsletter and redit also found it funny.

GoGraphviz: Go Parser for Graphviz's Dot Format.

picolang A functional programming language for working with pico services.
